Project Overview

This project is a fully responsive UsabilityHub-style landing page built using HTML5 and CSS3.
It showcases modern UI/UX design concepts, including reusable components, flexbox-based layouts, responsive typography, and structured sections for product features, examples, and CTA elements.

The website includes:

A top banner for announcements

Navigation bar with logo and menu links

Hero (header) section with call-to-action

Company logos section

Features overview section

Multiple large feature blocks

Examples gallery

CTA section

Footer with resource links

How to Run the Project

Download or clone the project folder.

Ensure the images folder is placed in the correct directory structure.

Open index.html in any browser.

No additional dependencies or build tools are required.

Customization

You can customize:

Colors inside :root {} in style.css

Text content in each section

Replace image sources inside the HTML

Add more feature cards or example items
